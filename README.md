# CloudExify Project 1 - Sales Data Analysis

## Project Overview
This project analyzes a Superstore sales dataset using Python. The dataset is cleaned, explored, and visualized to understand sales trends and business performance.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
- Sample Superstore Dataset

## Tasks Performed
- Loaded the dataset
- Checked dataset shape and columns
- Removed missing values
- Converted Order Date into DateTime format
- Calculated Mean, Median, and Mode of Sales
- Displayed descriptive statistics
- Found Top 10 Selling Products
- Analyzed Monthly Sales Trend
- Analyzed Sales by Region
- Analyzed Sales by Category
- Created Bar Charts and Line Plot

## Results
- Identified the top 10 products with the highest sales.
- Found monthly sales trends.
- Compared sales across different regions.
- Compared sales across different product categories.
